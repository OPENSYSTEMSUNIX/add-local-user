# add-user

Add the nessxsvc user

### Variables
| vaiable | vaule |
|---------|-------|
| `user:` | user to create |
| `comment:` | user comment |
| `optgroup:` | optional groups to add |
| `sudoers_cfg_file:` | /etc/sudoers|
| `sudoers_dropin_dir:` | /etc/sudoers.d |
| `sshkey:` | the ssh public key to copy in place |
